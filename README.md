## 📌 1. Project Overview
The **Interactive Personal Data Collector** is a foundational Python terminal application designed to interactively gather, process, and analyze personal user inputs. It serves as a comprehensive demonstration of core Python concepts by taking basic user demographics (Name, Age, Height, and a Favourite Number) and executing mathematical transformations, type casting, and deep memory analysis using built-in Python inspection tools. 

The application is structured to give real-time insights into how data types behave in computer memory, making it an excellent utility tool for validating data processing flows and understanding Python's core memory address management mechanisms (`id()` and `type()`).

---

## 🚀 2. Features
- **Interactive User Prompting:** Real-time data gathering using robust console-based input methods.
- **Dynamic Math Calculations:** Automates calculation of exact birth year, height conversion from meters/units to centimeters, and arithmetic properties (sum and product) of multiple inputs.
- **Explicit Type Casting:** Demonstrates smooth explicit type conversion across `string`, `integer`, and `float` data types.
- **Low-Level Memory Inspection:** Dynamically displays the internal system memory location (`id()`) and exact class definition (`type()`) of active variables.
- **String Formatting & Concatenation:** Implements modern formatted string literals (f-strings) and standard concatenation techniques to deliver clear summary blocks.
- **Formatted Tabular Summary:** Presents variables in a clean, professional, and well-aligned console data table for structured reading.
